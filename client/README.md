# 訂房網站 React App

## 簡介
這是一個使用 React 開發的訂房網站專案，提供用戶查詢、預訂房間的功能。網站包含列表頁面、房間詳情頁面和預訂確認頁面等。

## 功能
- 瀏覽飯店列表
- 查看房間詳情
- 預訂房間
- 用戶註冊和登入
- 管理用戶預訂記錄
